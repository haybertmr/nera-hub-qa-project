# NERA HUB QA Testing Project

QA testing project for the NERA HUB web application.

Application under test:
https://nera-hub.vercel.app/

## Testing Scope

This project includes the following QA activities:

- Exploratory testing
- Functional testing
- Bug reporting using Jira
- Test scenario design
- Test case design
- Automation testing
- Performance testing

## Tools Used

- Jira
- GitHub
- Selenium WebDriver
- Serenity BDD
- Cucumber
- Apache JMeter

## Repository Structure

test-plan → testing strategy  
test-scenarios → high level scenarios  
test-cases → detailed test cases  
bug-reports → documented bugs  
test-evidence → screenshots and proof  
automation → automated tests  
performance → load testing scripts
