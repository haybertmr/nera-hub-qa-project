# nera-hub-qa-project
QA testing project for NERA HUB web application including functional testing, bug reporting and automation.
NERA HUB QA Testing Project

This repository contains the QA testing process for the NERA HUB web application.

Testing types included:
- Exploratory testing
- Functional testing
- Bug reporting using Jira
- Test case design
- Automation with Selenium
- Performance testing with JMeter
